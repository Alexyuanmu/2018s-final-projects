## Project Abstract

​	C语言规范被ISO标准定义，但实际中C语言的实现与存在的代码的对C语言的理解存在歧义，与ISO标准不同。文章中作者通过问卷调查调研实际应用中C语言的不同参与者对C语言的理解，对C语言中指针与内存操作的语义歧义进行详细讨论。同时作者提出一种C语言的规范化模型Cerberus，通过对底层内存模型的修改，可以应用于不同的规范化模型，可作为小规模样例的测试集。
